## Usage

**First, You need have telegram account!**  

Add [@BotFather](https://telegram.me/BotFather) and send `/newbot` command to make new Bot, then you will get bot token.  

Setting token in `config.js`, then running:
```sh
$ npm install
$ npm start
```
